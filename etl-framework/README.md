# ETL Framework

This repository contains the specifications, configuration, agents, skills, and templates
that define the ETL framework.

## Structure

- `specs/` — Architecture, Frameworks, Components, Standards, Testing, Deployment, Governance, and AI generation specs
- `config/` — Configuration specs (source, pipeline, workflow, validation)
- `agents/` — Agent definitions for orchestration and pipeline layers
- `skills/` — Reusable "skill" definitions for common generation tasks
- `templates/` — Boilerplate templates for notebooks, workflows, dimensions, etc.

See `specs/README.md` for details on the specs folder.
